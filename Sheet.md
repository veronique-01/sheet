Git cheat Sheet
===============

##Initialisation et clonage
| Commande      | Description
|
|---------------|---------------------------
----------------------------------------|
|'git init'     | Initialise un nouveau depot Git dans le repertoire.
|
| 'Git clone[url]'| clone un depot distant sur votre machine locale.
|
